# 📖 Guia de Lançamento e Controle Dinâmico - Andrea Vargas

Este guia explica como usar os "Links Inteligentes" do site para as campanhas de tráfego pago da Mentoria. Você pode controlar o que o site exibe apenas mudando o link que você usa no Instagram.

---

## 1. O Link Normal (Site Portfólio)
Use este link para o dia a dia, na bio do Instagram ou para clientes de cabelo.
> **URL:** `https://seusite.com`
> *   **O que acontece:** O site abre completo, com menu, rodapé e todas as informações visíveis.

---

## 2. O Link de Campanha (Modo Foco / Lançamento)
Use este link nos anúncios da **Mentoria**.
> **URL:** `https://seusite.com/?mentoria=true`
> *   **O que acontece:** O site vira uma "Sala de Cinema". Esconde tudo e deixa apenas o vídeo. O botão VIP só aparece quando o vídeo termina.

---

## 3. Como trocar o VÍDEO (Sem mexer no código)
Se a Andrea gravou um vídeo novo, basta ela pegar o **ID do Vídeo** no YouTube e colocar no link.

**Como achar o ID:** É o código de letras e números que vem depois do `v=` no link do YouTube. 
*Exemplo:* Em `youtube.com/watch?v=dQw4w9WgXcQ`, o ID é `dQw4w9WgXcQ`.

> **URL com Vídeo Novo:** `https://seusite.com/?mentoria=true&v=COLOQUE_O_ID_AQUI`

---

## 4. Como trocar o GRUPO de WhatsApp (Sem mexer no código)
Se o grupo do WhatsApp lotar e ela precisar trocar para um novo, basta colocar o link ou o código do grupo no final.

> **URL com Grupo Novo:** `https://seusite.com/?mentoria=true&g=LINK_DO_GRUPO`

---

## 🚀 Modelos Prontos para Copiar (Exemplos)

**Link para Lançamento com vídeo específico:**
`https://seusite.com/?mentoria=true&v=dQw4w9WgXcQ`

**Link para Lançamento com vídeo e grupo específico:**
`https://seusite.com/?mentoria=true&v=dQw4w9WgXcQ&g=https://chat.whatsapp.com/ExemploLinkGrupo`

---

### 💡 Dicas Importantes:
*   **Sempre use `?mentoria=true` primeiro** antes de adicionar o `&v=` ou `&g=`.
*   O site fará o "Autoplay" (rodar o vídeo sozinho) no celular se o link tiver o `?mentoria=true`.
*   Qualquer dúvida, verifique se não há espaços nos links.
